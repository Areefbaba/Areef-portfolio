# Areef Baba Portfolio

Premium ATS-optimized portfolio for Areef Baba, built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, and lucide-react.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Sections

- Hero
- About
- Skills
- Experience
- Projects
- Certifications
- Education
- GitHub Stats
- Resume Download
- Contact
